# movies-repository
Proyecto enfocado en practicar tareas de ingeniería de software, como la gestión de entornos virtuales en Python y el desarrollo de una aplicación web, utilizando un dataset de películas para aplicar conceptos de análisis de datos.

Se analiza la informacion de un conjunto de datos de peliculas para determinar la popularidad segun el genero al que pertenecen, a traves de la creacion de graficos de dispersion e histogramas.
